SQL

MySQL

MySQL is an open-source relational database management system (RDBMS) that uses Structured Query Language (SQL) for accessing and managing databases,It is : Open Source ,Cross-Platform Support and High Performance

Numeric Data Types:

  • INT (integer):used to store Numbers without fractions
  • FLOAT/DOUBLE:Used to store decimal numbers
  • DECIMAL: Used to store decimal numbers

String Data Types

  • CHAR: A fixed-length string
  • VARCHAR: A variable-length string
  • TEXT: A large variable-length string

Date and Time Data Types

  • DATE: Stores a date value in the format "YYYY-MM-DD"
  • DATETIME: tores a date and time value in the format "YYYY-MM-DD HH:MM:SS"
  • TIME:Stores a time value in the format "HH:MM:SS"
  • YEAR:Stores a year value

Primary Keys

value which is unique and not null

Null and Not Null Constraints:

  • NULL: constraint allows a column to contain NULL values.
  • NOT NULL: constraint ensures that a column cannot have a NULL value

AUTO_INCREMENT

attribute in MySQL is used to generate a unique identifier for new rows in a table
